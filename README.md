## âš™ï¸ Setup & Installation

### Prerequisites
- Java 21 or higher
- Node.js (v18+)
- MySQL Server

### 1. Database Setup
1. Create a database named `restaurent_db` (or as configured in `application.properties`).
2. The tables will be automatically generated by Hibernate on the first run.

### 2. Backend Setup
```bash
cd RestaurentManagement_BE
./mvnw clean install
./mvnw spring-boot:run
```

### 3. Frontend Setup
```bash
cd RestaurentManagement_FE
npm install
npm start
```
